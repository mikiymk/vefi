# 演算子

- 値の計算をするもの。
- 二項演算子と前置単項演算子について扱う。

## `+`

### `整数 + 整数`

- 2つの数値を足し合わせる。
- 整数のオーバーフローは*不正動作*になる。

### `浮動小数点数 + 浮動小数点数`

- 2つの数値を足し合わせる。

### `複数要素ポインター + 整数`

- ポインターの位置をずらす。

### `Cポインター + 整数`

- ポインターの位置をずらす。

### `整数ベクトル + 整数ベクトル`

- 2つのベクトルの各要素を足し合わせる。
- 整数のオーバーフローは*不正動作*になる。

### `浮動小数点数ベクトル + 浮動小数点数ベクトル`

- 2つのベクトルの各要素を足し合わせる。

### `複数要素ポインターベクトル + 整数ベクトル`

- ベクトルの各ポインターの位置をずらす。

### `Cポインターベクトル + 整数ベクトル`

- ベクトルの各ポインターの位置をずらす。

## `+%`

### `整数 +% 整数`

- 2つの数値を足し合わせる。
- 整数のオーバーフローはラッピング演算になる。

### `整数ベクトル +% 整数ベクトル`

- 2つのベクトルの各要素を足し合わせる。
- 整数のオーバーフローはラッピング演算になる。
